# Retro Browser Games

A collection of retro-style browser games built with vanilla HTML5, CSS, and JavaScript.

## Games

### Tic-Tac-Toe
A classic two-player tic-tac-toe game with score tracking.

### Shooter
A top-down arcade-style shooter game featuring:
- Procedural pixel art sprites
- Web Audio API synthesized 8-bit sound effects
- Mouse-aimed click-to-shoot gameplay
- Arrow key movement
- Progressive difficulty across 5 levels
- Multiple enemy types with AI behavior
- Particle effects and screen shake

## Playing

Open either `tic-tac-toe.html` or `shooter.html` directly in a web browser. No build process or external dependencies required.

## Controls

### Shooter
- **Move**: Arrow keys or WASD
- **Aim**: Mouse
- **Shoot**: Left mouse button
- **Menu/Restart**: Click or Space/Enter
